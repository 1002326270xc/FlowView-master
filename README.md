**属性:**
```xml
<declare-styleable name="LineFlowLayout">
        <!--规定行数-->
        <attr name="flow_line_count" format="integer" />
        <!--文本是否居中-->
        <attr name="is_line_center" format="boolean" />
</declare-styleable>
```
